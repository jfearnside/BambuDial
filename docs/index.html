Setup After Flashing

  1. M5Dial boots and shows "BambuDial Setup" on the display
  2. Connect your phone or laptop to BambuDial-Setup WiFi
  3. A config page opens automatically (or go to http://192.168.4.1)
  4. Enter your WiFi, printer details (IP, serial, access code), and timezone
  5. Click Save & Reboot — your dial starts monitoring!
Find printer details on the LCD: Settings → Network → LAN Access Code & IP Address. Or check Bambu Studio's config file for all printers at once.

Features

  • Real-time print progress with color-coded arc
  • Nozzle & bed temperatures with heating/cooling icons
  • AMS filament type, color, and humidity
  • Print stage detail (leveling, heating, calibrating...)
  • Auto-rotate between active printers
  • ETA & time remaining (alternating)
  • 900+ error codes with human-readable messages
  • Local MQTT or Bambu Cloud connection

Hardware

You need an M5Stack M5Dial — a compact ESP32-S3 device with a 1.28" round display and rotary encoder.

M5Stack Store (~$25) · Amazon · AliExpress

Supported Printers

Works with any Bambu Lab printer with local MQTT:

X1C X1E X1 P1P P1S P2S A1 A1 Mini

Printers do NOT need to be in "LAN Only" mode. Local MQTT works alongside normal cloud connectivity.
